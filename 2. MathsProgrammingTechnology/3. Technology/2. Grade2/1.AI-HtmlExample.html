Welcome to My First HTML Page

HTML is the backbone of the web. It uses **opening and closing tags** to describe content. This text is in a paragraph tag (<p>).

Key Elements

  • <head>: Contains meta-information (like the page title).
  • <body>: Contains all visible content.
  • <a>: Used for creating hyperlinks. Search Google
  • <img>: Used to embed images.

This page is styled using CSS inside the <style> tags to make it look clean.

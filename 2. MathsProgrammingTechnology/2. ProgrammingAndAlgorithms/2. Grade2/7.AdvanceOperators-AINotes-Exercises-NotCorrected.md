Advanced Python Operators: Logic, Precedence, and Properties
============================================================

This guide extends the basic understanding of bitwise and boolean operators by incorporating logic simplification, operator precedence, and formal verification using truth tables.

1. Comprehensive Operator Precedence

------------------------------------

When multiple operators appear in a single expression, Python evaluates them in a specific hierarchy. Use parentheses `()` to override this order.

| Rank | Operator                                     | Description                                 |
| ---- | -------------------------------------------- | ------------------------------------------- |
| 1    | `()`                                         | Parentheses (Grouping)                      |
| 2    | `**`                                         | Exponentiation                              |
| 3    | `+x`, `-x`, `~x`                             | Unary Plus/Minus, Bitwise NOT               |
| 4    | `*`, `/`, `//`, `%`                          | Multiplication, Division, Floor Div, Modulo |
| 5    | `+`, `-`                                     | Addition, Subtraction                       |
| 6    | `<<`, `>>`                                   | Bitwise Left and Right Shifts               |
| 7    | `&`                                          | Bitwise AND                                 |
| 8    | `^`                                          | Bitwise XOR                                 |
| 9    | `\|`                                         | Bitwise OR                                  |
| 10   | `==`, `!=`, `>`, `<`, `>=`, `<=`, `is`, `in` | Comparisons, Identity, Membership           |
| 11   | `not`                                        | Logical NOT                                 |
| 12   | `and`                                        | Logical AND                                 |
| 13   | `or`                                         | Logical OR                                  |
| 14   | `=`, `+=`, `-=`, etc.                        | Assignments (Right-to-Left associativity)   |

2. Boolean Algebra Properties

-----------------------------

These properties allow us to simplify complex logic statements. Let $A$, $B$, and $C$ be boolean values.

### Fundamental Properties

* **Closure**: The result of a boolean operation is always a boolean (`True` or `False`).

* **Commutative**: $A \text{ and } B = B \text{ and } A$ | $A \text{ or } B = B \text{ or } A$

* **Associative**: $(A \text{ and } B) \text{ and } C = A \text{ and } (B \text{ and } C)$

* **Distributive**:
  
  * $A \text{ and } (B \text{ or } C) = (A \text{ and } B) \text{ or } (A \text{ and } C)$
  
  * $A \text{ or } (B \text{ and } C) = (A \text{ or } B) \text{ and } (A \text{ or } C)$ _(Unlike regular math, OR distributes over AND!)_

### Simplification & Absorption

* **Identity**: $A \text{ and True} = A$ | $A \text{ or False} = A$

* **Null (Domination)**: $A \text{ and False} = \text{False}$ | $A \text{ or True} = \text{True}$

* **Idempotent**: $A \text{ and } A = A$ | $A \text{ or } A = A$

* **Inverse (Complement)**: $A \text{ and (not } A) = \text{False}$ | $A \text{ or (not } A) = \text{True}$

* **Absorption**: $A \text{ or } (A \text{ and } B) = A$ | $A \text{ and } (A \text{ or } B) = A$

* **Double Negation**: $\text{not (not } A) = A$

* **De Morgan's Laws**:
  
  * $\text{not } (A \text{ and } B) = (\text{not } A) \text{ or } (\text{not } B)$
  
  * $\text{not } (A \text{ or } B) = (\text{not } A) \text{ and } (\text{not } B)$
3. Verifying Logic with Truth Tables

------------------------------------

A Truth Table lists all possible inputs to prove if two expressions are equal.

**Example: Verifying** $A \text{ and } (A \text{ or } B) = A$ **(Absorption)**

| A   | B   | (A or B) | A and (A or B) | Result    |
| --- | --- | -------- | -------------- | --------- |
| T   | T   | T        | T              | Matches A |
| T   | F   | T        | T              | Matches A |
| F   | T   | T        | F              | Matches A |
| F   | F   | F        | F              | Matches A |
